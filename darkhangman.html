Dark Hangman

Hangman

“A game in which the object is for one player to guess the letters of an unknown word before the player who knows the word creates a stick figure of a hanged man by drawing one line for each incorrect guess.” - Merriam Webster

App Features

  • Reward user with score for each consecutive successful guess of a word.
  • Keep maximum high score on local database.
  • User can't guess the same letter twice, and will be penalize for mistake.

    • Description

    Developed for a final project in a mobile softwar development course level 7 at Unitec, we were given 3 weeks to develop a working mobile application for android and i decided to make a horror theme game since it was close to halloween when i was developing the app and a hangman game just seems to be the right choice.
